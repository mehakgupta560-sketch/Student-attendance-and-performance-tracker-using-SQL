PROJECT TOPIC: Student Attendance & Performance Tracker USING SQL 
Problem Statement 
Educational institutes need a centralized system to track student attendance, academic 
performance, and eligibility efficiently. Manual tracking leads to errors, data redundancy, 
and poor insights. 
This project designs a relational database system that: 
• Tracks student details 
• Records daily attendance 
• Stores subject-wise marks 
• Analyses performance & attendance 
• Automates alerts for low attendance 
• Ensures data integrity & security
